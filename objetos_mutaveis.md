# Presentation 
Topic: Mutable objects in Python
## Introduction: What are mutable objects?
Mutables objects are those whose internal values can be altered. 
The meaning of the word "Mutable" is the ability to modify or edit a value.
Examples: Lists and dictionarys.

## Development 
### Lists, what they are?
Lists are data structures that store a sequence os values. 
1. They do not have a physical size.
2. they are considered as mutable objects, that is, we can add or remove elements from this list.
### What are they for?
It serves to store a sequence of values that can represents the number of monthly sales of a store.

### Dictionary, what they are?
A dictionary is a data structure that allows storing data in pairs, so each element consist of a key and a value associated with that key.
